## Overview

Tech Stack analysis identifies the technologies, platforms, and tools that companies use based on job descriptions, technical documentation, and public statements.

## Technology Categories

- **Cloud Platforms**: AWS, Azure, GCP usage
- **Databases**: SQL, NoSQL, data warehouses
- **DevOps Tools**: CI/CD, monitoring, orchestration
- **Languages**: Programming languages and frameworks
- **Security**: Security tools and platforms

## Use Case

<Tip>
Use tech stack data to:
