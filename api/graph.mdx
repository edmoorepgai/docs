## Get Company Graph

Retrieve the knowledge graph for a specific company. The graph maps relationships, relevance signals, and cross-references between the company and related entities in your workspace.

This endpoint calls PG:AI's relevance engine to compute real-time relationship data.

<ParamField path="company_id" type="string" required>
  The unique identifier (UUID) of the company. Must be a valid UUID format.
</ParamField>

**Permission:** `insights:read`

<RequestExample>
```bash cURL
curl -X GET "https://api.getpg.ai/v1/companies/{company_id}/graph" \
  -H "x-api-key: your_api_key"
```
</RequestExample>

<Note>

</Note>

### Validation

The `company_id` must be a valid UUID. Requests with invalid formats will receive a `400` error:

```json
{
  "error": "Company ID must be a valid UUID"
}
```
